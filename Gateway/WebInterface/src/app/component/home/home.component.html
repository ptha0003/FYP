RFID localization using IoT Network


This is a web interface built for my indoor localization using RFID and IoT network FYP. The web interface has two main functions, first is to let the users register their RFID tag unique ID to the corresponding user details. The second function of this web interface is to display a table of all users that have been detected. The table displays the following data first and last name, contact information such as email and phone number, the rooms they have been in as well as the entry and exit time.

The web interface is communicating with a MongoDB database. The MongoDB Database has two collections, one for storing the registered users and another collection for the detected users. When registering a user the information registered is stored in the registered user collection and any detected users will be stored in the other collection, where the display table will collect information from and display accordingly.
